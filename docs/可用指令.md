## 七、Cursor 用量

| 指令示例 | 说明 |
|----------|------|
| **cursor用量** / 查询cursor用量 | 调用 Cursor Dashboard 聚合用量 API，返回账户用量数据。 |
| **cursor今日用量** / 查询cursor今日用量 | 调用 Cursor Dashboard 今日筛选用量 API，返回当天用量数据。 |
| **同步cursor登录态** | 自动读取本机 Chrome 中 cursor.com 登录 Cookie，并注入当前服务内存。 |

**接口地址：**

`https://cursor.com/api/dashboard/get-aggregated-usage-events`

`https://cursor.com/api/dashboard/get-filtered-usage-events`

**认证方式：**

- 支持配置 `CURSOR_API_TOKEN`（推荐）或 `CURSOR_COOKIE`。
- 未配置时，执行 `cursor用量` 会自动尝试同步本机 Chrome 登录态 Cookie。
- 可选覆盖接口地址：`CURSOR_USAGE_API_URL`。
- 请求方式为 `POST`，请求体包含：
  - `teamId`（默认 `-1`，可用 `CURSOR_TEAM_ID` 覆盖）
  - `startDate`（按每月固定锚点时间计算；锚点可用 `CURSOR_USAGE_MONTHLY_BASE_START` 配置，默认 `1774764813000`）
- 请求会携带来源校验头（默认）：
  - `Origin=https://cursor.com`
  - `Referer=https://cursor.com/dashboard`
  - 如需覆盖，可配置 `CURSOR_API_ORIGIN`、`CURSOR_API_REFERER`
